# :material-calendar-edit: Book

Here you will find how to book a class/meeting

!!! note
    Students should book on weekends for one to two in advanced, new comers allowed
    to book only **one session** in this period

!!! warning
    Canceling an appointment is allowed and costs you **30%** of a single session payment,
    for example if a session was $10, if you cancel it, 30% == $3 is taken and $7 is refunded


## Test

For book a test session please kindly refer to this like [Book a meeting with shakiba](https://book.shakiba.net/)

!!! info
    You can book only **one session**, more then one will be rejected


## Classes

For book classes session please kindly refer to this like [Book a meeting with shakiba](https://book.shakiba.net/)

!!! info
    You can book for **two weeks** in advanced according to our plan and your payment amount
